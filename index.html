Difference between JavaScript & HTML

    JavaScript:-

  • JavaScript is an Imperative Language:-
    In Imperative Language, we teach the computer what to do.

    HTML:-

  • HTML is a Declarative Language:-
    In Declarative Language, we tell the computer what to do.

JSX

JSX is a language developed by Facebook engineers in the year 2011. JSX stands for JavaScript XML.
It is a Declarative Language and we teach the computer what to do. It is a combination of JavaScript and HTML.

React-Native

React is a JavaScript library used for making user interfaces.
React-Native is a framework for building native apps using React.

Components

Every React Application is made up of nested components.
We first imported 3 components namely <View>, <Text> and <Button>.

Then we should create a class and export it. In the class we have a render function.

Whatever component we return to the render function, it will display it.
But it can return only one component.

If we need to render multiple components, we must nest them inside the <View> component.

Every react component must start in Capital letter.

We created our own component called <AnyColorButton>. This component renders a button with any color.
